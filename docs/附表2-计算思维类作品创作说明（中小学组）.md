# 附表2 "计算思维"类作品创作说明（中小学组）

| 项目 | 内容 |
|------|------|
| 所属类别 | □创新开发 £创意编程 □创意编程专项 |
| 作品名称 | AI学习助手 - 智能解题系统 |
| 创作思想（创作背景、目的和意义） | 背景：传统学习方式无法满足学生个性化需求，解题过程缺乏及时详细指导。目的：开发智能学习助手，通过多Agent架构实现题目智能分析和解答。意义：帮助学生理解解题过程，培养计算思维能力，提供个性化学习支持。 |
| 创作过程（运用了哪些技术或技巧完成主题创作，哪些是得意之处） | 技术：前端HTML5/CSS3/JavaScript，后端Flask/Python，多模态大模型（图片识别），DeepSeek（解题）。技巧：多Agent协作架构设计，流水线式处理算法，Prompt优化。得意之处：多模态输入支持，详细解题步骤生成，知识点自动分析，响应式界面设计。 |
| 原创部分 | 多Agent协作工作流设计，题目解析算法，解答生成Prompt模板，前端交互逻辑，历史记录功能。 |
| 参考资源（参考或引用他人资源及出处） | 前端库：marked.js、DOMPurify、KaTeX（CDN引用）；后端框架：Flask；AI模型接口文档。 |
| 制作使用的软件工具及运行环境 | 开发工具：Trae IDE、Python 3.12；运行环境：Flask Web服务，支持主流浏览器（Chrome、Firefox、Safari）。 |
| 其他说明（需要特别说明的问题） | 本作品使用了第三方AI模型API，需要网络连接才能正常运行。系统会存储解题历史以提供个性化学习支持。 |
